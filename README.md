# Introduction to Deep Learning 

An introduction book to deep learning for college sophomores with a decent high school math background and knowledge of Python. The book should be comprehensive. There should be some light math, and also there should be definitely the Python code. 

**Author:** Dr. Ernesto Lee 

## Chapters

1. From Neurons to Networks: Understanding the Building Blocks
2. Training Your First Neural Network: The Learning Process
3. Deep Dive into Deep Networks: Multi-Layer Architectures
4. Convolutional Neural Networks: Seeing Through AI Eyes
5. Recurrent Neural Networks: Understanding Sequences and Time
6. Optimization and Regularization: Making Networks Learn Better
7. Popular Architectures: Standing on the Shoulders of Giants
8. Real-World Applications: Putting It All Together
